# 🏨 Hotel Booking Cancellation Prediction

An end-to-end data analysis and machine learning project on 87,000+ hotel booking records — exploring what drives cancellations and building a predictive model.

## 📊 Live Dashboard
🔗 [Coming soon on Streamlit Cloud](#)

## 🔑 Key Findings
- 27.5% overall cancellation rate — ~24,000 bookings lost
- July–August drive 38% of annual revenue
- Transient customers cancel at 30.15% — 3x higher than Group bookings (9.80%)
- Repeat guests cancel 3.7x less than new guests (7.73% vs 28.33%)
- Non-refundable deposits show 94.7% cancellation rate — a counterintuitive data anomaly
- Long-term bookings cancel at 39.74% vs 18.63% for last-minute bookings
